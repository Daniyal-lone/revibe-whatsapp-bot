# Revibe Salon Automation

Full-stack CRM/POS automation for Revibe Unisex Salon.

## What is included

- Staff login with a simple PIN
- One-screen visit/payment entry
- Customer, staff, service, and transaction database
- Receipt PNG generation
- WhatsApp receipt sending hook
- Owner dashboard
- Automation queue view
- Time-to-return reminder engine
- Retry-safe receipt and marketing workers
- Render and Railway deployment files

## What you need from the client

1. Official WhatsApp Business number
2. Meta WhatsApp Cloud API access token
3. WhatsApp phone number ID
4. Public business name and address for receipts
5. Staff list
6. Service list with prices
7. Owner PIN and staff PIN
8. Deployment choice: Railway or Render

## Local setup

```bash
cp .env.example .env
docker compose up -d
npm install
npm run migrate
npm run seed
npm start
```

Open:

```text
http://localhost:3000
```

Default demo PINs from `.env.example`:

```text
Staff: 1111
Owner: 2222
Developer: 3333
```

## Deployment recommendation

Use Render free web service + Supabase free Postgres for the zero-cost MVP.

1. Push this folder to GitHub.
2. Open Render and deploy the connected repo.
3. Add the environment variables from `RENDER_ENVIRONMENT.md`.
4. Deploy once.
5. Copy the Render public URL.
6. Set `PUBLIC_BASE_URL` to that URL.
7. Redeploy.

`render.yaml` is included for Render.

## WhatsApp setup

Leave this off until the business number and templates are approved:

```text
WHATSAPP_ENABLED=false
```

When ready:

```text
WHATSAPP_ENABLED=true
WHATSAPP_PHONE_NUMBER_ID=...
WHATSAPP_ACCESS_TOKEN=...
PUBLIC_BASE_URL=https://your-live-domain.com
```

The receipt sender uses WhatsApp image messages with a public image URL generated by this app.
